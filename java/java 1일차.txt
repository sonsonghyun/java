5강 변수 
변수란? 값을 저장할 수 있는 메모리의 공간을 의미한다
자바는 강형(strongly-typed)언어 > 해당 변수에 어떤 데이터를 담을 것인지를 결정해야 함

변수는 
1. 값을 담을 수 있는 그릇이다
2. 선언할 때 무엇을 담을지 정한다
3. 정해진 형의 값만 담을 수 있다

변수의 관례
4. 변수는 항상 소문자로 시작합니다
5. 변수에 두 단어 이상을 선언하려면 두 번째 단어는 대문자로 시작해야 합니다

상수 > 변하지 않는 값 
코드 사용시 여러번 사용할 것 같은 값을 상수 처리함

final 형 이름;
final int J >> 상수명은 대문자로 사용함 

7강 기본형 
자바에서 변수를 사용하려면 자료형을 지정해줘야 한다
boolean isFun = true;
char c=’f’; >> 작은 따옴표를 사용해야 함
int x=59;
long big = 3456789L; >> 뒤에 소문자 l이나 대문자 L을 붙여줘야 함
float f =32.5f >> 뒤에 소문자f나 대문자 F를 붙여줘야 함
double d=23.34;

8강 기본형 형변환
큰 자료형에 들어있는 것을 작은 자료형에 넣으려고 하면 컴파일러는 에러를 발생시킴

작은 자료형의 값을 큰 자료형에 넣을 때는 자연스럽게 형변환이 일어난다
이런 경우를 묵시적(암묵적) 형변환이라고 한다

long x2=5;
int y2 = (int) x2;
System.out.println(y);
>>> 형변환을 하려면 값을 넣기전에 해당 형을 괄호 안에 넣어준다
>>> 이를 명시적(강제) 형변환 이라고 한다 (큰그릇을 작은 그릇에 담을 때)
