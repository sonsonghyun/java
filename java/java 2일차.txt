26강 메서드
자바는 객체지향 언어 
객체지향 언어란 하나의 사물을 하나의 클래스로 설명하는 언어
사물에 대한 설명은 그 물체의 상태와 행동으로 나눈다
물건의 상태에 해당하는 구성요소가 필드라면
물체의 행동에 해당하는 구성요소가 메서드이다

상태 = 필드 
행동 = 메서드

메서드는 입력값이 있고 그 입력값을 받아서 무언가를 한 다음에 결과를 도출해내는 
수학의 함수와 비슷한 개념

이 때, 입력값을 인자라고 하고 결과값을 리턴값이라고 한다
입력값은 매개변수 or 인자라고 한다
매개변수를 파라미터, 인자를 아규먼트 
매개변수는 전달된 인자를 받아들이는 변수를 의미
인자는 어떤 함수를 호출할 때 전달되는 값 자체를 의미
